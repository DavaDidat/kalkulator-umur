# Age Calculator (Day 13)

Project ini adalah aplikasi sederhana untuk menghitung umur pengguna berdasarkan tanggal lahir.

Dibuat sebagai bagian dari latihan roadmap frontend.

## Fitur

- Input tanggal lahir menggunakan **Flatpickr (JavaScript Datepicker)**
- Menghitung umur secara akurat menggunakan **Luxon**
- Menampilkan hasil umur dalam:
  - Tahun
  - Bulan
  - Hari
- Validasi jika tanggal lahir tidak diisi
- Tampilan sederhana, responsive, dan user-friendly
- Pesan error dengan animasi

## Teknologi yang Digunakan

- HTML
- CSS
- JavaScript
- Flatpickr
- Luxon

## Cara Menggunakan

1. Pilih tanggal lahir dari kalender
2. Klik tombol **Hitung Umur**
3. Hasil umur akan ditampilkan di layar

## Validasi

- Input tidak boleh kosong
- Tidak bisa memilih tanggal di masa depan
